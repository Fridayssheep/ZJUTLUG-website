# 关于我们
![ZJUTLUG](/img/logo.webp)

浙江工业大学Linux用户社团（`ZJUT Linux User Group`，简称ZJUT LUG）是由在校的 GNU/Linux爱好者发起并组成的全校性群众团体，是一个以推广自由软件文化、宣传自由软件价值为目的的协会组织。

我们以“开放、共享、协作、创新”为核心理念，致力于学习和研究Linux操作系统及其他开源软硬件技术，并通过多个高质量项目和活动推动开源精神的发展。

我们的社团成立于2025年4月。经过一个寒冬的筹备，在杭州初春的日子里，[mgg](https://github.com/MangoGovo)同学以一场出色的线上答辩开启了社团的征途。

我们有很多想弄弄的新项目。不管现在水平如何，如果你对技术感兴趣，愿意了解开源文化，我们都欢迎你加入。
