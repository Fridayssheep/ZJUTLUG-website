#  活动预告 | 浙江工业大学 ZJUT LUG “河畔义修”活动公告 

浙江工业大学 Linux User Group (ZJUT LUG) 将于本年**12月11日周四**举办“河畔义修”活动 ，面向全校师生提供免费的电脑软件技术支持与数字文化共享服务。

## 活动主题

本次活动主题为“电脑软件义诊与数字文化共享：系统优化与开源小游戏体验”。我们将提供免费、专业的电脑软件故障排查与修复服务 ，解决同学们遇到的系统运行缓慢、驱动异常、软件冲突等软件问题 ，并普及系统优化、安全配置和数据备份等实用技能 。同时，活动设置开源小游戏体验环节~~来吸引凑第二课堂学时的同学~~。

### 活动信息

| 项目 | 内容 |
| :--- | :--- |
| **时间** | **2025年12月11日 9:00 - 18:30** (白天全时段服务)  |
| **地点** | **上塘河畔** (户外场地)  |

### 服务内容概述

本次活动将提供系统优化、驱动安装、软件调试、病毒查杀等服务。现场将有技术服务人员专注于软件故障排除、系统优化和配置调试。另外，现场将设置专门的开源小游戏体验区，由工作人员引导寻求第二课堂学时的同学进行体验和打卡。

> **⚠️ 特别强调：** 本次义修服务**仅针对软件问题**提供，不处理任何硬件损坏或更换需求。

### 重要提醒：数据备份要求
为规避数据丢失风险，本次活动将**强制要求**学员在签到时完成重要数据备份。工作人员在操作前将再次口头提醒备份的重要性 。请计划参与义修的师生提前做好数据备份工作 。

欢迎大家届时前往上塘河畔参与体验与服务！
